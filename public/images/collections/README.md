# Collections Section Images

This directory should contain 4 collection category images for the homepage.

## Required Images

1. **marble-types.jpg** - Showcase of different marble types/varieties
2. **tile-collections.jpg** - Display of tile collections
3. **custom-orders.jpg** - Custom marble work or design examples
4. **premium-finishes.jpg** - Premium/luxury marble finishes

## Image Specifications

- **Format**: JPG or WebP
- **Resolution**: Minimum 800x800px (square format recommended)
- **Aspect Ratio**: 1:1 (square) works best for the grid layout
- **File Size**: Optimized for web (recommended: under 300KB per image)

## Image Sources

You can download free images from:
- **Unsplash**: https://unsplash.com (search: "marble", "tiles", "marble types")
- **Pexels**: https://pexels.com (search: "marble collection", "tile design")
- **Pixabay**: https://pixabay.com (search: "marble", "tiles")

## Notes

- All images should be properly licensed for commercial use
- Images will be automatically optimized by Next.js Image component
- A gradient fallback will be shown if images fail to load
- Images should showcase the product category clearly
