# Hero Section Images

This directory should contain 4 high-quality hero images for the carousel.

## Required Images

1. **hero-1.jpg** - Outdoor/Living Space with marble tiles
2. **hero-2.jpg** - Modern bathroom with marble/tile finishes
3. **hero-3.jpg** - Kitchen with marble countertops or tiles
4. **hero-4.jpg** - Living room showcasing marble flooring or wall tiles

## Image Specifications

- **Format**: JPG or WebP
- **Resolution**: Minimum 1920x1080px (Full HD) or higher
- **Aspect Ratio**: 16:9 or 21:9 (wide format)
- **File Size**: Optimized for web (recommended: under 500KB per image)

## Image Sources

You can download free images from:
- **Unsplash**: https://unsplash.com (search: "marble tiles", "marble flooring", "luxury marble")
- **Pexels**: https://pexels.com (search: "marble interior", "tile design")
- **Pixabay**: https://pixabay.com (search: "marble", "tiles")

## Notes

- All images should be properly licensed for commercial use
- Images will be automatically optimized by Next.js Image component
- A gradient fallback will be shown if images fail to load
